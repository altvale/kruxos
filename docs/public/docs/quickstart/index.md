# Quickstart

Get KruxOS running and your first agent connected in under 15 minutes.

## Choose your path

| Step | What you'll do | Time |
|------|---------------|------|
| 1. [Install](install.md) | Boot KruxOS via Docker (`altvale/kruxos`) or the VM image | 5 min |
| 2. Connect your model | [Claude Code](claude-code.md) / [Claude Desktop / API](connect-claude.md) / [OpenAI (+ Codex)](connect-openai.md) / [Gemini](connect-gemini.md) / [Local](connect-local.md) | 5 min |
| 3. Explore | [Dashboard](dashboard.md) or [CLI](cli.md) | 5 min |
| 4. (Optional) [Use email capabilities](gmail.md) | Adapters ship in v0.0.1; the operator-facing OAuth connection flow lands in v0.0.2 | 5 min |

!!! tip "Fastest path"
    Docker install + Claude (MCP-native, zero adapter code) gets you running in under 5 minutes.
