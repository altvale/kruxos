# Contact

## Enterprise inquiries

For enterprise pricing, evaluation access, and custom deployments:

**Email:** enterprise@kruxos.com

Please include:

- Your organization name
- Number of agents you plan to deploy
- Key use cases
- Any compliance requirements (SOC2, ISO27001, etc.)

We typically respond within 1 business day.

## Security

To report a security vulnerability, please email **security@altvale.com**. Do not file a public GitHub issue for security reports.

We follow responsible disclosure and aim to acknowledge reports within 24 hours.
